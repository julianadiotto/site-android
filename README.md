 Curiosidades de Tecnologia - O Mascote do Android
Este projeto é um site informativo detalhando a história da criação do mascote do Android, desde os primeiros conceitos ("Dandroids") até o icônico "Bugdroid". O site foi desenvolvido com foco em acessibilidade, semântica e adaptabilidade para diferentes tamanhos de tela.

 Funcionalidades e Destaques
Design Responsivo: Uso da tag <picture> para carregar imagens diferentes dependendo da largura da tela, otimizando o consumo de dados em dispositivos móveis.

Semântica HTML5: Utilização correta de tags como <header>, <nav>, <main>, <article>, <aside> e <footer> para melhor indexação em motores de busca (SEO).

Navegação Interna: Menu funcional com âncoras para seções específicas da página (Home, Notícias, Curiosidades, Fale Conosco).

Mídia Incorporada: Inclusão de vídeos do YouTube e uso da tag <abbr> para fornecer explicações sobre os nomes das versões do Android.

Identidade Visual: Estilização focada na temática tecnológica com suporte a links externos e tipografia limpa.

 Tecnologias Utilizadas
HTML5: Estrutura robusta e semântica.

CSS3: Estilização personalizada e consultas de mídia (Media Queries).

Componentes de Imagem Flexíveis: Implementação de artes adaptativas.

 O que aprendi neste projeto
Este projeto foi fundamental para praticar conceitos avançados de Front-End:

Hierarquia de Títulos: Organização lógica de <h1> a <h3>.

Uso de aside: Como separar conteúdos periféricos (como a lista de versões do Android) do fluxo principal do artigo.

Experiência do Usuário (UX): Garantir que links externos abram em novas abas (target="_blank") e que o conteúdo seja legível em qualquer dispositivo.
